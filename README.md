## 📌 소개

반복적인 엑셀 작업을 자동화하는 VBA 애드인입니다.  
생활기록부 작성, 데이터 정리, 셀 병합 등 교무업무에 필요한 기능을 모았습니다.

---

## 📥 설치 방법

### 1. 애드인 설치
1. `vbacode.xlam` 파일을 `C:\excel\` 폴더에 복사
2. Excel 실행 → `파일` → `옵션` → `추가 기능`
3. 하단 `관리: Excel 추가 기능` → `이동` 클릭
4. `찾아보기` → `C:\excel\vbacode.xlam` 선택

### 2. 리본 메뉴 적용
1. Excel 완전 종료
2. `C:\Users\{사용자명}\AppData\Local\Microsoft\Office` 이동
3. `CustomUI_final.xml` → `Excel.officeUI`로 이름 변경 후 덮어쓰기
4. Excel 재실행

## 🛠 기능 소개

### Delete (행 삭제)
| 버튼 | 기능 |
|------|------|
| **RowDelete** | 선택한 열에서 빈 셀이 있는 행 전체 삭제 |
| **Restore** | 마지막 삭제 작업 복원 |

### Merge (셀 병합)
| 버튼 | 기능 |
|------|------|
| **셀병합 가로** | 빈 셀을 왼쪽 셀과 가로 병합 |
| **셀병합 세로** | 같은 값이거나 빈 셀을 위쪽과 세로 병합 |

### Korean (한글 처리)
| 버튼 | 기능 |
|------|------|
| **단어 조사 연결** | 받침에 따라 은/는, 이/가, 을/를, 과/와 자동 선택 |
| **바이트 길이** | 생활기록부용 글자수 계산 (한글 3byte, 영문 1byte) |

### Calc (문자열 계산)
| 버튼 | 기능 |
|------|------|
| **CalStr** | 문자열에서 숫자와 수식 추출하여 계산 (예: "3+5원" → 8) |
| **CalStrSum** | 범위 내 모든 셀의 CalStr 결과 합계 |

### Fill (채우기)
| 버튼 | 기능 |
|------|------|
| **사용자지정목록** | 선택 영역을 자동채우기 목록으로 등록 |
| **가로붙여넣기** | 선택 영역을 가로 방향으로 붙여넣기 |
| **세로붙여넣기** | 선택 영역을 세로 방향으로 붙여넣기 |
| **오른쪽 채우기** | 병합 셀을 펼쳐서 오른쪽 열에 채우기 |
| **랜덤숫자** | 1~N까지 중복 없는 랜덤 숫자 생성 |

### Find (검색)
| 버튼 | 기능 |
|------|------|
| **팀원으로 팀명찾기** | TEXTJOIN+IF 수식으로 팀명 검색 |
| **이름찾기** | 조건부 서식으로 같은 값 강조 (노란색) |
| **중복찾기** | 중복 값을 연한 노란색으로 표시 |
| **해제하기** | 배경색 + 조건부 서식 모두 해제 |

## 📁 파일 구성

```
C:\excel\
├── vbacode.xlam          # 애드인 파일
└── (이미지 파일들)        # RowDelete 설명용

%LOCALAPPDATA%\Microsoft\Office\
└── Excel.officeUI        # 리본 메뉴 설정
```

### VBA 모듈 구조
| 모듈 | 기능 |
|------|------|
| `Mod_Delete` | 행 삭제, 백업/복원 |
| `Mod_Merge` | 셀 병합 |
| `Mod_Korean` | 한글 조사, 바이트 계산 |
| `Mod_Calc` | 문자열 수식 계산 |
| `Mod_Fill` | 채우기, 붙여넣기 |
| `Mod_Find` | 검색, 중복 찾기 |
